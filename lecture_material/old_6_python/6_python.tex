    \section{Coding is Just Coding}\label{coding-is-just-coding}

Once you learn one coding language, you can read (and sometimes even
write) in a lot of other coding languages. Today we're going to go
through how to code what we did on the first day in Python (another
popular coding language). The syntax is slightly different, but you can
still read a lot of it. The point of this is to realize that if someone
hands you a code script in a different language, you shouldn't freak
out.

In R Studio, create a python script. This will require the
\texttt{retriculate} package and you may also need to install the
\texttt{IRkernel} package, as well. Install both.

    \section{Advanced}\label{advanced}

I wrote this is in a Jupyter Notebook, which is the pythons verion of an
R notebook. To write a Jupyter Notebook using R, you need to do a few
aditional (and advanced) steps.

In an R consol, run
\texttt{install.packages(\textquotesingle{}IRkernel\textquotesingle{})}

In the Terminal, run \texttt{jupyter\ notebook}

This will open up a Jupyter Notebook host in your web browser. You can
create a new .ipynb file (aka a Jupyter Notebook).

Unlike an R Markdown file, you cannot automatically knit it to an HTML
or pdf document. However, you can download it as an HTML, or you can
download a LaTex file and compile the LaTex file to a PDF on your
computer (which is a more advanced step then downloading the HTML).
